# Интернет магазин

Описание: интернет магазин для просмотра каталогов

Перед запуском необходимо установить [NodeJs](https://nodejs.org/en/)

## Установка пакетов
```
npm install
```

### Запуск в режиме разработки
```
npm run serve
```

### Сборка проекта
```
npm run build
```

### Json server install and start
```
npm i -g json-server
json-server -w db/db.json -p 3456
```
